# WhereIsMyCursor

Windows 系統托盤游標位置指示器 - 透過動態箭頭圖示幫助你快速找到滑鼠游標。

![托盤圖示](doc/images/tray-icon.png)

## 功能特色

- **動態箭頭指示**：系統托盤圖示實時顯示指向游標位置的箭頭
- **距離顏色指示**：10 級漸層顏色 (紅色=近 → 綠色=遠) 表示游標距離
- **閃爍提示**：近距離時箭頭會閃爍，越近閃越快
- **可調整更新頻率**：低 (2 FPS)、中 (10 FPS)、高 (30 FPS)
- **位置校正**：可調整 X/Y 偏移值以精確指向托盤圖示位置

## 系統需求

- Windows 10/11
- .NET 8 Runtime

## 安裝

### 從原始碼建置

```bash
cd src/WhereIsMyCursor
dotnet build -c Release
```

建置完成後，執行檔位於 `bin/Release/net8.0-windows/WhereIsMyCursor.exe`

## 使用方式

1. 執行程式後會自動最小化至系統托盤
2. 托盤圖示會顯示指向游標的箭頭
3. 右鍵點擊托盤圖示可開啟選單：
   - **設定...** - 開啟設定視窗
   - **結束** - 關閉程式

## 設定選項

![設定視窗](doc/images/settings.png)

| 設定項目 | 說明 |
|----------|------|
| 啟用閃爍效果 | 開啟/關閉近距離閃爍提示 |
| 更新頻率 | 低 (省電) / 中 (建議) / 高 (流暢) |
| X 偏移 | 調整托盤圖示水平位置校正 |
| Y 偏移 | 調整托盤圖示垂直位置校正 |

## 距離指示

| 距離 | 顏色 | 閃爍 |
|------|------|------|
| 0-400px | 紅色 | 快閃 (150ms) |
| 400-1200px | 橘色系 | 快閃 (200-300ms) |
| 1200-2400px | 黃色系 | 中閃 (300-400ms) |
| 2400-3600px | 綠色系 | 慢閃 (500-1000ms) |
| >3600px | 亮綠色 | 不閃爍 |

## 技術規格

- 框架：.NET 8 + WPF
- 托盤圖示：System.Drawing 動態生成 16x16 圖示
- 游標位置：Win32 API (GetCursorPos)
