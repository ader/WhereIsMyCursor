# WhereIsMyCursor

Windows 系統托盤游標位置指示器 - 透過動態箭頭圖示幫助你快速找到滑鼠游標。

![托盤圖示](doc/images/tray-icon.png)

## 功能特色

- **動態箭頭指示**：系統托盤圖示實時顯示指向游標位置的箭頭
- **距離顏色指示**：10 級漸層顏色 (紅色=近 → 綠色=遠) 表示游標距離
- **閃爍提示**：近距離時箭頭會閃爍，越近閃越快
- **可調整更新頻率**：低 (2 FPS)、中 (10 FPS)、高 (30 FPS)
- **位置校正**：可調整 X/Y 偏移值以精確指向托盤圖示位置

## 系統需求

- Windows 10/11
- .NET 8 Runtime

## 安裝

### 從原始碼建置

```bash
cd src/WhereIsMyCursor
dotnet build -c Release
```

建置完成後，執行檔位於 `bin/Release/net8.0-windows/WhereIsMyCursor.exe`

## 使用方式

1. 執行程式後會自動最小化至系統托盤
2. 托盤圖示會顯示指向游標的箭頭
3. 右鍵點擊托盤圖示可開啟選單：
   - **設定...** - 開啟設定視窗
   - **結束** - 關閉程式

## 設定選項

![設定視窗](doc/images/settings.png)

| 設定項目 | 說明 |
|----------|------|
| 啟用閃爍效果 | 開啟/關閉近距離閃爍提示 |
| 更新頻率 | 低 (省電) / 中 (建議) / 高 (流暢) |
| X 偏移 | 調整托盤圖示水平位置校正 |
| Y 偏移 | 調整托盤圖示垂直位置校正 |

## 距離指示

| 級別 | 距離範圍 | 箭頭顏色 | 閃爍間隔 |
|------|----------|----------|----------|
| 0 | 0 - 400px | 紅色 (#FF0000) | 150ms (快閃) |
| 1 | 400 - 800px | 紅橘色 (#FF2D00) | 200ms (快閃) |
| 2 | 800 - 1200px | 橘紅色 (#FF5A00) | 200ms (快閃) |
| 3 | 1200 - 1600px | 橘色 (#FF8700) | 300ms (快閃) |
| 4 | 1600 - 2000px | 橘黃色 (#FFB400) | 300ms (中閃) |
| 5 | 2000 - 2400px | 黃色 (#FFE100) | 400ms (中閃) |
| 6 | 2400 - 2800px | 黃綠色 (#D4F000) | 500ms (慢閃) |
| 7 | 2800 - 3200px | 淺綠色 (#A0FF00) | 700ms (慢閃) |
| 8 | 3200 - 3600px | 綠色 (#6CFF00) | 1000ms (極慢) |
| 9 | ≥ 3600px | 亮綠色 (#48FF00) | 不閃爍 |

閃爍方式：非對稱透明度切換 (80% 顯示 / 20% 暗淡)

## 技術規格

- 框架：.NET 8 + WPF
- 托盤圖示：System.Drawing 動態生成 16x16 圖示
- 游標位置：Win32 API (GetCursorPos)
