# WhereIsMyCursor 開發藍圖

本文件列出未來可能的功能改善與開發計畫。

---

## 功能改善

### 高優先級

- [ ] **開機自動啟動選項**
  - 在設定視窗加入「開機時自動執行」核取方塊
  - 實作方式：寫入 Windows Registry `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`

### 中優先級

- [ ] **托盤圖示 Tooltip**
  - 顯示目前游標與托盤的距離數值
  - 格式：「距離: 1234px」

- [ ] **多語系支援**
  - 加入英文介面
  - 使用 .resx 資源檔管理多語系字串

### 低優先級

- [ ] **自訂顏色**
  - 讓使用者自訂近距離/遠距離的顏色
  - 在設定視窗加入顏色選擇器

---

## 發佈與安裝

### 高優先級

- [ ] **安裝程式**
  - 建立 MSIX 或 MSI 安裝包
  - 包含開始選單捷徑
  - 包含解除安裝功能

### 中優先級

- [ ] **程式碼簽章**
  - 取得程式碼簽章憑證
  - 簽署執行檔與安裝程式
  - 避免 Windows SmartScreen 警告

### 低優先級

- [ ] **自動更新**
  - 啟動時檢查 GitHub Release 是否有新版本
  - 提示使用者下載更新

## 程式碼品質

### 中優先級
