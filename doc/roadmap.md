# WhereIsMyCursor 開發藍圖

本文件列出未來可能的功能改善與開發計畫。

---

## 功能改善

### 已完成

- [x] **開機自動啟動選項** (v1.0.1)
  - 在設定視窗加入「開機時自動執行」核取方塊
  - 實作方式：寫入 Windows Registry `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`

- [x] **多語系支援** (v1.1.0)
  - 加入英文介面
  - 使用 .resx 資源檔管理多語系字串
  - 設定視窗語言切換選項

### 中優先級

- [ ] **自訂顏色**
  - 讓使用者自訂近距離/遠距離的顏色
  - 在設定視窗加入顏色選擇器

- [ ] **全域熱鍵**
  - 按下熱鍵時觸發游標尋找效果 (箭頭飛向游標)
  - 預設熱鍵：Ctrl + Alt + F
